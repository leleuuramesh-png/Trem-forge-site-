Visão geral

A Trem Forge orquestra agentes de IA especializados — planejamento, front-end, back-end, banco de dados e deploy — para levar uma ideia descrita em linguagem natural até um software funcional, publicado e pronto para uso.

Esta página documenta os principais pontos de integração do site: o widget de chat com IA, o formulário de envio de projeto e a estrutura de páginas do site institucional.

Começando

O site é composto por HTML estático (sem build step), com os seguintes arquivos principais:

  • index-2.html — landing page principal
  • templates.html — galeria de exemplos de projetos
  • docs.html — esta página
  • privacidade.html e termos.html — páginas legais
  • images/ — imagens usadas nos planos de fundo

Para rodar localmente, basta servir a pasta com qualquer servidor estático (não há dependências de build).

API do chat de IA

O widget "Testar IA" na landing page envia mensagens para uma função serverless:

POST /.netlify/functions/chat
Content-Type: application/json

{
  "messages": [
    { "role": "user", "content": "Quero um app de delivery" }
  ]
}

Resposta esperada:

{
  "reply": "Ótimo! Me conta mais sobre..."
}

Essa função já está implementada em netlify/functions/chat.js e usa a API da Anthropic (Claude) por trás. Para funcionar em produção, configure a variável de ambiente abaixo no painel da Netlify (Site settings → Environment variables):

ANTHROPIC_API_KEY=sk-ant-sua-chave-aqui

# opcional — padrão: claude-sonnet-4-6
ANTHROPIC_MODEL=claude-sonnet-4-6

Sem essa chave configurada, a function responde com erro 500 e o widget mostra uma mensagem de indisponibilidade — o restante do site continua funcionando normalmente.

Formulário de projeto (Formspree)

O formulário "Enviar Projeto" já está funcional e envia os dados via POST para o Formspree, sem necessidade de backend próprio.

Stack e deploy

  • HTML/CSS/JS puro — sem framework de build
  • Fontes via Google Fonts (Geist, Inter, JetBrains Mono)
  • Formulário de leads via Formspree
  • Analytics via Google Tag (gtag.js)
  • Hospedagem recomendada: Netlify (para suportar as Functions do chat)

Perguntas frequentes

Preciso saber programar para usar a Trem Forge?

Não. A plataforma foi criada justamente para quem não é desenvolvedor — você descreve o que quer e os agentes cuidam da parte técnica.

Posso pedir alterações depois que o projeto for gerado?

Sim, novas instruções podem ser enviadas a qualquer momento para ajustar o que já foi construído.
